# irritool

The **irritool** package provides computational tools in R for
irrigation.

## Installation

You can install the development version of irritool from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("joaobtj/irritool")
```

## Documentation and Tutorials

Detailed examples, including the complete workflow for calculating crop
water requirements and simulating the daily soil water balance, are
available in the package vignettes.

If you are using the package website, you can read the full tutorials in
the Articles section: <https://joaobtj.github.io/irritool/>
